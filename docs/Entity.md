# 创建实例

1、最简单的实例：

- 只包含实值属性，也就是类型为 boolean, integer, long, string, float, double 的属性；
- 属于叶子实例，只被别人引用，而不引用任何其他实例；
- 唯一性问题的原则：当且仅当所有实值属性相同时，可视为同一实例；
